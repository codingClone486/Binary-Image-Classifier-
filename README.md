Binary Image Classifier (Deep Learning) 🧠📷

A deep learning powered web application that performs binary image classification using a trained convolutional neural network.
The application allows users to upload an image and receive a prediction generated by the model in real time.

The project is built with TensorFlow/Keras and deployed using Streamlit with an interactive user interface

🚀 Features

- Upload images for classification
- Deep learning based prediction
- Interactive web interface
- Real-time inference
- Clean dark-themed UI
- User feedback system
- Fast image preprocessing and prediction

🧠 Model Overview

The classifier is built using TensorFlow and Keras and trained for binary image classification.

Key aspects of the model:

- Convolutional Neural Network (CNN)
- Image preprocessing and normalization
- Binary classification output
- Efficient inference for real-time prediction

🛠️ Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow (PIL)

⚙️ Installation

Clone the repository:

https://github.com/codingClone486/Binary-Image-Classifier-.git
cd binary-image-classifier

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

📸 How It Works

1. Upload an image using the interface.
2. The image is resized and normalized.
3. The trained model processes the image.
4. The application outputs the predicted class.

Project Evolution

This project represents the initial stage of building an image classification system using deep learning.

A more advanced version of this project has been developed as **VisionaryAI**, which expands the model to perform multi-class image classification across a large number of object categories.
